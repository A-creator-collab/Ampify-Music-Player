# 🎵 Ampify Music Player

Ampify is a modern desktop music player built using Python, Tkinter, MySQL, and Pygame.

## ✨ Features

- User Authentication
- Music Library
- Playlist Management
- Favorites
- Song Queue
- Recently Played
- Search Songs
- Dark & Light Theme
- Volume Control
- Progress Bar
- Playlist Playback

---

## 🛠 Technologies Used

- Python
- Tkinter
- Pygame
- MySQL
- PyMySQL
- bcrypt
- Mutagen
- ttkthemes

---

## 📚 Data Structures Used

- Hash Table
- Queue (Deque)
- Quick Sort
- Linear Search

---

## 🗄 Database

MySQL

Tables:

- Users
- Songs
- Playlists
- Playlist Songs
- Favorites

---

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/YourUsername/Ampify-Music-Player.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run

```bash
python main.py
```
